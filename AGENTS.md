# fm-reboot vault — точка входа для AI-агентов

> Canonical instruction set для любых AI-агентов (Claude Code, Cursor, Codex, Cline),
> работающих в этом vault'е и связанных репозиториях. Имя `AGENTS.md` — IDE-нейтральная
> конвенция; рядом лежит `CLAUDE.md` (однострочный редирект сюда для Claude Code).

Ты работаешь в персональной памяти проекта **reboot** (Obsidian + Git с авто-pull/commit/push).
Этот файл — точка входа; полный контракт по импорту ниже.

## Обязательно к загрузке

@meta/CLAUDE.md

Этот файл — правила работы с vault'ом (identity, чек-лист после работы, inbox-протокол,
verify-gate, хуки, задачи). Имеет дублёра `meta/AGENTS.md` (редирект на него же).

## Если зашёл впервые

1. `meta/welcome.md` — введение для людей.
2. `meta/CLAUDE.md` (выше) — твой контракт.
3. `state/now.md` — что происходит сейчас.
4. `meta/index.md` — карта vault'а.

## Контекст по требованию (читай явно, когда нужно)

- **`state/now.md`** — снимок состояния. Читай в начале работы над задачей.
- **`meta/index.md`** — карта vault'а. Читай при навигации.
- **`rules/*.md`** — security / verify-gate / vault-git / task-discipline. Читай по релевантности.
