---
description: Проверить непрочитанные сообщения в inbox/<handle>.md
---

1. Определи handle: `git -C <vault> config kb.author`.
2. Открой `inbox/<handle>.md`. Найди строки с маркером 🟡 (непрочитанные заголовки сообщений).
3. Нет 🟡 — скажи «входящих нет» и закончи.
4. Есть 🟡 — обработай каждое по протоколу из [[meta/CLAUDE|meta/CLAUDE.md]] §«Inbox-протокол»:
   - информационное → упомяни, 🟡 → ✅;
   - вопрос/мнение → ответь через `/msg @<sender>`, 🟡 → ✅;
   - запрос на ревью → прочитай документ, напиши ревью в `state/daily/<today>.md`, `/msg @<sender>`, 🟡 → ✅;
   - write-side commitment в shared knowledge / разрушительное действие → спроси через `AskUserQuestion`.
5. Замену 🟡 → ✅ делай через Edit, дописывая короткую пометку об обработке.
6. В конце — строка: «Inbox: обработано N (✅), действия: ...».
